# Portfolio - Nguyễn Thị Thùy Linh 🌟

Trang web Portfolio cá nhân hiện đại, chuyên nghiệp dành cho Thực tập sinh Marketing. Được xây dựng với hiệu ứng mượt mà và giao diện tối ưu.

## 🚀 Giới thiệu
Đây là dự án Portfolio cá nhân của **Nguyễn Thị Thùy Linh**, sinh viên năm 4 ngành Quản trị kinh doanh. Trang web được thiết kế để giới thiệu bản thân, các kỹ năng chuyên môn trong lĩnh vực Marketing (Content, Social Media, Video Production) và các kinh nghiệm làm việc thực tế.

## ✨ Tính năng nổi bật
- **🌓 Chế độ Sáng/Tối (Dark Mode)**: Tự động chuyển đổi giao diện theo sở thích người dùng.
- **🎨 Hiệu ứng Motion**: Sử dụng Framer Motion cho các hiệu ứng xuất hiện, cuộn trang và thanh tiến trình (Progress Bar) mượt mà.
- **📱 Thiết kế Responsive**: Giao diện hiển thị hoàn hảo trên mọi thiết bị (Mobile, Tablet, Desktop).
- **📄 Tải CV trực tiếp**: Tích hợp nút tải CV nhanh chóng.
- **⚡ Tốc độ cực nhanh**: Xây dựng trên nền tảng Vite và React 19.

## 🛠️ Công nghệ sử dụng
- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion (motion/react)](https://www.framer.com/motion/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 📦 Hướng dẫn cài đặt và chạy

1. **Clone dự án**:
   ```bash
   git clone <link-github-cua-ban>
   cd <ten-thu-muc>
   ```

2. **Cài đặt các thư viện cần thiết**:
   ```bash
   npm install
   ```

3. **Chạy ứng dụng ở môi trường phát triển (Local)**:
   ```bash
   npm run dev
   ```
   Mở trình duyệt và truy cập: `http://localhost:3000`

4. **Xây dựng bản Production**:
   ```bash
   npm run build
   ```

## 📁 Cấu trúc thư mục chính
- `src/App.tsx`: Chứa toàn bộ logic và giao diện chính của Portfolio.
- `src/index.css`: Cấu hình Tailwind CSS và các biến màu sắc.
- `public/`: Chứa ảnh đại diện (`profile.jpg`) và file CV (`CV_NguyenThiThuyLinh.pdf`).
